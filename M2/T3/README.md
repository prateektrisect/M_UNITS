## Problem Statement

Given a string as input replace 1 by A, 2 by b, 3 by c and 4-9 by x. 0 and other
characters appear as it is in the new string and are not changed.

**Sample Test Case**
```
Input:
12049brt

Output:
Ab0xxbrt
```
